# Éclat Doréa Collection Logo 设计方案

## 品牌分析
- **品牌名称**: Éclat Doréa collection
- **法语含义**: "金色光芒系列"
- **定位**: 高端黄金珠宝
- **目标客户**: 追求奢华、品质的消费者
- **关键词**: 优雅、光芒、金色、精致、法国奢华

## Logo设计选项

### 选项1: 经典豪华版
```
ÉCLAT DORÉA
collection
```
- **字体**: Cormorant Garamond (衬线体)
- **颜色**: 深金色渐变 (#D4AF37 → #FFD700)
- **特点**: 经典法式优雅，上下布局
- **适合**: 正式场合，品牌主logo

### 选项2: 简约现代版
```
ÉD
collection
```
- **字体**: Montserrat (无衬线体)
- **颜色**: 纯金色 (#CA8A04)
- **特点**: 首字母缩写，简约现代
- **适合**: 小尺寸应用，页脚logo

### 选项3: 艺术装饰版
```
Éclat Doréa
✦ collection ✦
```
- **字体**: Playfair Display (艺术衬线体)
- **颜色**: 深褐色+金色 (#1C1917 + #CA8A04)
- **特点**: 添加钻石符号装饰
- **适合**: 包装、宣传材料

### 选项4: 徽章组合版
```
[冠冕图标]
Éclat Doréa
collection
```
- **字体**: Cinzel (装饰衬线体)
- **颜色**: 深色背景+金色文字
- **特点**: 结合皇冠图标，贵族感
- **适合**: 品牌徽章，水印

## Logo规格要求

### 文件格式
- **主Logo**: PNG 400×120px (透明背景)
- **页脚Logo**: PNG 200×60px
- **Favicon**: PNG 32×32px
- **备用格式**: SVG (矢量格式)

### 颜色方案
- **主金色**: #CA8A04 (温暖金色)
- **深金色**: #D4AF37 (传统金色)
- **亮金色**: #FFD700 (明亮金色)
- **深褐色**: #1C1917 (搭配背景)

### 字体选择
1. **Cormorant** - 经典法式优雅
2. **Playfair Display** - 艺术装饰风格
3. **Cinzel** - 贵族徽章风格
4. **Montserrat** - 现代简约风格

## 设计原则
1. **奢侈品感**: 使用金色和深色搭配
2. **法式优雅**: 体现Éclat Doréa的法语渊源
3. **珠宝关联**: 暗示光芒、精致、价值
4. **可读性**: 在不同尺寸下清晰可读
5. **一致性**: 与Shopify主题设计系统匹配

## 实施计划
1. 创建多个logo变体
2. 生成PNG和SVG格式
3. 优化文件大小
4. 上传到Shopify主题
5. 测试不同位置的显示效果
